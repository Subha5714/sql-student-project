# SQL Student Database Project

This project demonstrates basic SQL operations using a student database.

## Features:
- Created a database and table using SQL
- Inserted student records
- Retrieved data using SELECT queries
- Applied filtering using WHERE clause
- Used GROUP BY for aggregation
- Performed conditional queries for data analysis

## Tools Used:
- MySQL

## Description:
This project focuses on SQL-based data handling and query execution. It is a simplified database version of a Student Result Management System, demonstrating core database concepts and operations.
